Ejercicio Planteado

Crear una pagina con varios tabas, cada uno mostrando contenido diferente utilizando Boostrap para estilizar los tabs y JavaScript para controlar el cambio de tabs y mostrar el contenido correspondiente.


INDICACIONES

En micro grupos de 3 integrantes, utilizando Bootstrap deberá crear un sitio web en donde cree las siguientes páginas:

  • 1. Home -> incluirá la portada con los datos generales de la asignatura, la tarea e integrantes.
  • 2. Cada integrante deberá crear una página personal en donde incluya su Curriculum Vitae
  • 3. Una página que contendrá la solución al ejercicio seleccionado por el equipo (será entregado durante clase teórica)
  • 4. Una página de conclusiones
  • 5. El sitio deberá estar publicado en GITHUB pages

Control de tabas con JS

De esta manera se soluciono el problema de controlar el cambio de los tabs y mostrar el contenido de cada uno de ellos

Creaciond de Constantes

const optionCont1 = document.getElementById("optionCont1");
const optionCont2 = document.getElementById("optionCont2");
const optionCont3 = document.getElementById("optionCont3");

Se capturaron las etiquetas con id opciones las cuales son las tabs


Contenido de las tabs

//contenido de las tabas
const cont1 = document.getElementById("cont1");
const cont2 = document.getElementById("cont2");
const cont3 = document.getElementById("cont3");

Se capturaron las etiquetas con id cont1 las cuales son el contenido de las tabs


Funcion mostrar contenido

        
          let chose = 1;
  
          const changeOption = () => {
              chose == 1 ? 
                  optionCont1.classList.value = 'nav-link active',
                  cont1.classList.value = 'container tab-pane active'
              ) 
              : (
                  optionCont1.classList.value = 'nav-link ',
                  cont1.classList.value = 'container tab-pane'
              )
  
              chose == 2 ? (
                  optionCont2.classList.value = 'nav-link active',
                  cont2.classList.value = 'container tab-pane active'
              ) 
              : (
                  optionCont2.classList.value = 'nav-link ',
                  cont2.classList.value = 'container tab-pane'
              )
  
              chose == 3 ? (
                  optionCont3.classList.value = 'nav-link active',
                  cont3.classList.value = 'container tab-pane active'
              ) 
              : (
                  optionCont3.classList.value = 'nav-link ',
                  cont3.classList.value = 'container tab-pane'
              )
            
          }

Ejecucion de la funcion al clickear el tab correspondiente

        
        //evento al hacer click en las tabs
        optionCont1.addEventListener('click', ()=>{
            chose = 1
            changeOption()
        })

        optionCont2.addEventListener('click', ()=>{
            chose = 2
            changeOption()
        })

        optionCont3.addEventListener('click', ()=>{
            chose = 3
            changeOption()
        })
